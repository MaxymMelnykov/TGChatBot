callbacks.py

callbacks.setup_callbacks(bot)

Реєструє всі обробники команд і callback-запитів для Telegram-бота.

Ця функція налаштовує основну логіку обробки взаємодії з користувачем:

  • Обробляє команду /start та виводить стартове меню.

  • Обробляє callback-запити: - Перехід по головному меню, допомога, контакти, FAQ. - Налаштування користувачів (замовник/ЖК). - Вибір типу та матеріалу контейнерів. - Опція датчиків наповнення для підземних контейнерів. - Підрахунок потрібних контейнерів для ЖК. - Запит номера телефону для замовника.

Працює в парі з іншими модулями проєкту: config, handlers, Container, markups, user_data, utils.

Параметри:

bot – Екземпляр TeleBot для взаємодії з Telegram API.
