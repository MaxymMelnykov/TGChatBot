user_data.py

user_data.user_data = {}

Словник за замовчуванням для зберігання даних користувачів.

Використовує defaultdict з лямбда-функцією, яка ініціалізує кожного користувача з набором значень за замовчуванням.

user_data.user_type

Тип користувача, наприклад, „замовник“ або „представник ЖК“.

Type:

str or None

user_data.container_calc_res_ra

Результат розрахунку для контейнера.

Type:

int

user_data.container_volume_of_all_orders

Загальний об’єм всіх замовлених контейнерів.

Type:

int

user_data.area

Площа, яку обслуговує користувач.

Type:

str or None

user_data.apartments

Кількість квартир, пов’язаних з користувачем.

Type:

str or None

user_data.container_name

Назва контейнера.

Type:

str or None

user_data.container_type

Тип контейнера.

Type:

str or None

user_data.container_material

Матеріал контейнера.

Type:

str or None

user_data.container_quantity

Кількість контейнерів.

Type:

int

user_data.container_underground_sensor

Наявність сенсора у підземного контейнера.

Type:

bool

user_data.container_width

Товщина стінки контейнера.

Type:

int

user_data.telephone_number

Номер телефону користувача.

Type:

str or None

user_data.orders

Список замовлень користувача.

Type:

list

user_data.total_sum

Загальна сума всіх замовлень користувача.

Type:

float

Повертає:

Об’єкт defaultdict, який автоматично ініціалізує нові записи з вказаними значеннями за замовчуванням.

Тип повернення:

defaultdict
